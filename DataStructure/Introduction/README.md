# 演算法要點

1. 演算法的定義:為完成某一特定工做，所設計出的一組有限、有序的指令集合
2. 演算法須符合:
   - 輸入(不一定需要)
   - 輸出(必須有)
   - 明確性(每個步驟需有明確的定義，不能含糊不清)
   - 有限性(在有限的步驟之後一定能終止)
   - 有效性(AL 的每一指令必須是基本且可以有效執行的指令)

# ADT(Abstract Data Type)

## 三要素

1. 一個特定的 Domain
2. 一個運算的集合
3. 一些描述運算的 axioms
